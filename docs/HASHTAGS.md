# كتالوج الوسوم

يوضح هذا الملف جميع الوسوم التي يتعرّف عليها البوت أثناء إدخال المواد، مع ترتيبها والقواعد الخاصة بكل نوع.

## القواعد العامة

- يجب كتابة كل وسم في سطر مستقل ويبدأ بعلامة `#`.
- يُسمح بوسم محتوى واحد فقط من وسوم المحتوى المذكورة أدناه.
- الترتيب مهم في الأنواع التي تُحدَّد أدناه؛ مخالفة الترتيب أو نقص الوسوم المطلوبة ينتج عنه رسالة رفض مناسبة.

## الوسوم العامة

### وسم المحاضرة
`#المحاضرة_1: عنوان` أو `#محاضرة_الأولى: عنوان` لتحديد رقم المحاضرة وعنوانها.

رسالة الرفض عند عدم توفير رقم أو عنوان:
```
هذا النوع يتطلب وسم محاضرة بصيغة: #المحاضرة_1: <العنوان>
```

### وسم السنة
`#1446` أو `#1446هـ` لتمييز السنة (بين 1300 و1600).

### وسم المحاضر
يبدأ أحد البادئات التالية ثم الاسم (`_` يمثل مسافة):
`#الدكتور_`, `#الدكتورة_`, `#الأستاذ_`, `#الأستاذة_`, `#المهندس_`, `#المهندسة_`, `#م_`, `#م`.

## وسوم الأقسام

| الوسم | القسم |
|---|---|
| `#نظري` / `#theory` | قسم نظري |
| `#مناقشة` / `#مناقشه` / `#discussion` | قسم مناقشة |
| `#عملي` / `#lab` | قسم عملي |
| `#رحلة` / `#field_trip` | قسم رحلات ميدانية |

| الوسم | البطاقة |
|---|---|
| `#التوصيف` | التوصيف |
| `#المفردات_الدراسية` | المفردات الدراسية |
| `#الواقع_التطبيقي` | الواقع التطبيقي |

| `#مهارات` | مهارات |
| `#مشاريع_مفتوحة_المصدر` | مشاريع مفتوحة المصدر |

### مثال خاطئ
استخدام وسم بطاقة متبوعًا بوسم محتوى آخر مثل:
```

#slides
```
الرسالة المتوقعة:
```
رجاءً اتّبع ترتيب الوسوم لهذا النوع: #التوصيف
```

- **الحضور**: `#جدول_الحضور`، `#الحضور`، `#جدول_الغياب`، `#attendance`
- **الخطة الدراسية**: `#الخطة_الدراسية`، `#الخطة`، `#خطة_الدراسة`، `#study_plan`
- **روابط القنوات**: `#روابط_القنوات`، `#القنوات`، `#قنوات`، `#channels`
- **مخرجات التعلم**: `#مخرجات_التعلم`، `#مخرجات`، `#نتائج_التعلم`، `#outcomes`
- **نصائح الدراسة**: `#نصائح`، `#نصيحة`، `#study_tips`، `#tips`
- **أفكار مشاريع**: `#مشاريع`، `#المشاريع`، `#project_ideas`، `#projects`
- **برامج**: `#برامج`، `#البرامج`، `#software`، `#programs`
- **تطبيقات**: `#تطبيقات`، `#التطبيقات`، `#applications`، `#apps`
- **مهارات**: `#مهارات`، `#المهارات`، `#competencies`، `#skills`
- **منتديات**: `#منتديات`، `#المنتديات`، `#communities`، `#forums`
- **مواقع**: `#مواقع`، `#المواقع`، `#websites`، `#sites`
- **المفردات الدراسية**: `#المفردات_الدراسية`، `#المفردات_الدرسية`، `#المفردات`، `#study_vocab`
- **الواقع التطبيقي**: `#الواقع_التطبيقي`، `#الواقع`، `#real_world`، `#practical`
- **مراجع**: `#مراجع`، `#المرجع`، `#references`، `#refs`
- **مشاريع مفتوحة المصدر**: `#مشاريع_مفتوحة_المصدر`، `#مشاريع_مفتوحة`، `#open_source_projects`، `#oss_projects`
- **محتوى آخر**: `#محتوى_اخر`، `#محتوى_آخر`، `#miscellaneous`، `#misc`

يجب استخدام وسم واحد فقط من هذه الوسوم في الرسالة.

### مثال خاطئ للحضور
```

```
الرسالة المتوقعة:
```
رجاءً اتّبع ترتيب الوسوم لهذا النوع: #جدول_الحضور
```

### شرائح / صوت / فيديو
هذه الوسوم تتطلب وسم محاضرة لكن لا تشترط ترتيبًا محددًا.
مثال صحيح:
```
#audio
#المحاضرة_2: تحويل لابلاس
```
مثال خاطئ بدون وسم محاضرة:
```
#audio
```
الرسالة المتوقعة:
```
هذا النوع يتطلب وسم محاضرة بصيغة: #المحاضرة_1: <العنوان>
```

### أنواع أخرى
`#summary`، `#notes`، `#related`، `#external_link`، `#simulation`، `#mind_map`، `#transcript` وغيرها لا تملك ترتيبًا خاصًا، لكن يجب الالتزام بوسم محتوى واحد لكل رسالة.
